# Alura-Midi 🎵
Um mini MIDI com alguns sons de instrumentos (ESTUDO)

## Finalizado ✔️

## Descrição do projeto 📋
Projeto concluído e construido durante estudos de JavaScript 
![tela-alura-midi](https://user-images.githubusercontent.com/92035697/158249707-3ba7242c-8328-46c1-a8cc-837bb43c5e5b.png)

## 🔨 Funcionalidade do projeto
- `Funcionalidade 1`: Poder tocar sons de instrumentos ao clicar com o mouse
- `Funcionalidade 2`: Poder tocar sons de instrumentos com o teclado "tab" ou "espaço"

## 🛠️ Construído com
* [HTML](https://developer.mozilla.org/pt-BR/docs/Web/HTML) - Estrutura e elementos da página
* [CSS](https://developer.mozilla.org/pt-BR/docs/Web/CSS) - Estilização dos elementos
* [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) - Linguagem utilizada para lógica dos elementos
